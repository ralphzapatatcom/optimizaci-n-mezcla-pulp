# Optimización de Mezcla de Alimentos con PuLP 🌽🥦

Este proyecto utiliza **Programación Lineal** para encontrar la mezcla de ingredientes de menor costo que cumpla con requisitos nutricionales específicos.

## 🚀 Tecnologías utilizadas
* **PuLP**: Para el modelado y resolución del problema de optimización.
* **NumPy**: Para la manipulación de matrices de nutrientes.
* **Matplotlib**: Para la visualización de la composición óptima y distribución de costos.

## 📊 Cómo funciona
El script calcula la proporción ideal de Maíz y Soya para minimizar el gasto, asegurando que se alcancen los niveles mínimos de proteína y fibra.
